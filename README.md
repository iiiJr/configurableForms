# configurable_forms
表单自定义配置化项目，基于Vue.js, Vuex, antd。实现了表单自定义配置，JSONEdit中可编辑表单中的配置，预览表单

演示地址：https://configurable-forms-rhzb94e6s-iiijr.vercel.app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
